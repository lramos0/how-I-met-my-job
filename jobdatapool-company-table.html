JobDataPool Company Table

This page explains company-level records in JobDataPool, a free and public datalake of job data related tables.

Datalake overviewJobs tableSkills table

Core fields

  • Company canonical name and normalized web domain
  • Industry, size band, and maturity indicators
  • Hiring momentum derived from active postings
  • Cross-reference keys to jobs and skills tables

How Hiring Cafe uses this table

Hiring Cafe uses company rows to power employer summaries, forum groupings, and context-rich search experiences.
