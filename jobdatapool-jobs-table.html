JobDataPool Jobs Table

This page documents the jobs table in JobDataPool, a free and public datalake of job data related tables.

Datalake overviewCompany tableSkills table

Core fields

  • Job title, company name, and canonical location
  • Posted date, ingest date, and freshness markers
  • Employment type, seniority, and industry labels
  • Apply URL, source URL, and quality score signals

How Hiring Cafe uses this table

Hiring Cafe uses jobs table rows for searchable listing rails, mobile filters, direct apply actions, and freshness sorting.
